Project Description

What is SheaMart and what does it do?

What is SheaMart?

SheaMart is a secure online retail order management system built as part of our Capstone group project. It allows customers to browse products, add items to a cart, place orders, and track their delivery status in real time.

Admins can manage the product catalogue, view all customer orders, update order status, and monitor the business through a statistics dashboard.

The system is built with security at its core — every user has a role, and access is strictly controlled based on that role.

Our Module — Order Tracking

Our group is responsible for the Order Tracking Module. This module allows customers to see exactly where their order is at every stage of the delivery process.

Order statuses include:

  • 📬 Order Placed — Order received by the system
  • ⚙️ Processing — Order being prepared
  • 🚚 Shipped — Order on its way
  • ✅ Delivered — Order received by customer

Only admins can update order status. Customers can only view their own orders.

Problem Statement

The problem our system solves.

❌ The Problem

  • Customers have no visibility of their order after placing it
  • Admins update order status manually on paper or spreadsheets
  • No secure login — anyone can access sensitive customer data
  • Customers must call the store just to find out where their order is
  • No separation between admin and customer access levels

✅ Our Solution

  • Real-time order tracking — customers see status instantly
  • Admins update status through a secure web interface
  • Secure login with bcrypt password hashing
  • Customers only see their own orders — never others
  • Role-based access control — Admin vs Customer
