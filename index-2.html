System Modules

Each module handles a key part of the retail management system.

Product Catalogue

Browse products. Admins add and edit items; users can only view.

Shopping Cart

Add or remove products. Each user can only access their own cart.

Order Processing

Place and validate orders. All inputs are checked before saving.

User Management

Order Tracking

View all users, orders, and key statistics. Admin access only.

Access Control — Role Permissions

Security is enforced at every route. No role can exceed its permissions.
